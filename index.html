Palindrome in JAVA

Palindrome

is a word, number, phrase, or other sequence of characters which reads the same backward as forward, such as madam, racecar.

Here you will learn how to validate or know if a word or number are Palindrome using the programming language JAVA

Here's the main code for compare to words and know if are Palindromes : 

                
 public static String reverseWord(String word)
 {
    String reverseWord = "";
    for (int i = word.length() - 1; i >= 0; i--) 
    {
        reverseWord += word.charAt(i);
    }
        
    return reverseWord;
 }
